业务部-销售流程与客户管理规范

一、销售流程

1.1 线索获取

销售线索来源包括：
- 线上渠道：公司官网咨询、微信公众号、抖音企业号、百度推广
- 线下渠道：行业展会、客户推荐、合作伙伴介绍
- 主动开发：电话营销、邮件营销、上门拜访

所有线索必须在 CRM 系统中建档，记录来源渠道、联系人信息、需求描述。

1.2 线索分配

- 线上线索由系统自动按区域分配给对应销售人员
- 线下线索由销售总监根据销售人员的行业经验和当前负荷手动分配
- 销售人员收到线索后2小时内必须完成首次联系

1.3 客户跟进

客户分级标准：
- A级客户（重点客户）：年采购额50万以上，每周至少跟进1次
- B级客户（潜力客户）：年采购额10-50万，每两周至少跟进1次
- C级客户（普通客户）：年采购额10万以下，每月至少跟进1次

每次跟进必须在 CRM 中记录：沟通方式、沟通内容、客户反馈、下一步计划。

1.4 报价与合同

报价流程：
1. 销售人员根据产品定价体系和客户需求编制报价单
2. 折扣在10%以内由销售主管审批
3. 折扣在10%-20%由销售总监审批
4. 折扣超过20%需总经理审批

合同签订注意事项：
- 合同条款必须经法务审核
- 付款方式优先选择预付或分期付款
- 违约金条款应对等
- 明确交付标准、验收流程、售后责任

二、客户管理

2.1 客户信息管理

客户档案包含以下信息：
- 基本信息：公司名称、地址、统一社会信用代码
- 联系人信息：姓名、职务、手机、微信、邮箱
- 业务信息：合作历史、采购记录、合同清单
- 跟进记录：沟通历史、需求变更、投诉处理

2.2 客户满意度管理

每季度进行一次客户满意度调查：
- 产品质量满意度（1-5分）
- 交付及时性满意度（1-5分）
- 售后服务满意度（1-5分）
- 整体服务满意度（1-5分）

满意度低于3分的客户，销售经理需在3个工作日内回访并制定改善方案。

2.3 客户流失预警

出现以下情况时，系统自动触发流失预警：
- 连续3个月无采购记录
- 最近一次沟通超过30天
- 客户满意度评分连续两次下降
- 客户投诉未在48小时内解决

预警触发后，销售总监需在24小时内制定挽留方案。

三、售后协调

3.1 售后问题分级

- P0（紧急）：影响客户正常经营，1小时内响应，4小时内解决
- P1（重要）：功能不可用但有替代方案，2小时内响应，24小时内解决
- P2（普通）：非关键功能问题，8小时内响应，72小时内解决

3.2 售后流程

1. 客户通过客服热线/微信/邮件提交问题
2. 客服记录问题并分级
3. 技术团队进行问题诊断
4. 解决方案实施
5. 客户确认问题解决
6. 问题归档并纳入知识库

四、销售业绩考核

4.1 考核指标

- 销售额完成率（权重40%）
- 回款率（权重20%）
- 新客户开发数（权重20%）
- 客户满意度（权重10%）
- 老客户复购率（权重10%）

4.2 提成制度

- 完成月度目标80%以下：提成比例3%
- 完成月度目标80%-100%：提成比例5%
- 完成月度目标100%-120%：提成比例7%
- 完成月度目标120%以上：提成比例10%，超出部分额外奖励

五、商务礼仪

1. 拜访客户前必须预约，不得临时上门
2. 商务洽谈穿着正装或商务休闲装
3. 准时赴约，迟到必须提前通知
4. 不得在客户面前贬低竞争对手
5. 不得向客户做超出公司授权范围的承诺
6. 商务宴请标准：人均不超过200元
